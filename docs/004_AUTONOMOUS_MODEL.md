# Autonomous Model

OSForge otonomi modeli, ajanlarin sinirli ve izlenebilir yetkilerle calismasini esas alir.

Baslangic modeli:

- Niyet kullanicidan gelir.
- Ajan plan uretir.
- Ajan kucuk adimlarla uygular.
- Sonuclar raporlanir ve gerektiginde insan onayi istenir.

Tam otonomi yerine guvenilir kademeli otonomi hedeflenir.
